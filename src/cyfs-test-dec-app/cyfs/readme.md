## 更新cyfs-ts-sdk 依赖的版本
```
npm run init source  //切换源码
npm run init cyfs-node //切换自己编译的sdk
npm run init cyfs-sdk-nightly //切换npm nightly版本
npm run init cyfs-sdk //切换npm beta版本
```
