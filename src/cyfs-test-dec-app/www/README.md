# Technology stack

TypeScript + React + Antdesign

# www directory structure

```
-apis // Interface folder, storing request-related packages
-assets // resource folder, storing static resource files, pictures, etc.
-components // public components folder
    -Hello
        -Hello.tsx
        -Hello.module.less
-constants // folder to store constants
-containers // container components folder
-hooks // public hooks folder
-i18n // internationalization language pack
-layout // layout template component
-pages // pages folder
-routers // route folder
-stores // global state store folder
-styles // global or common style file
-types // global generic types file
-utils // Store the packaged utility function folder
initialize.ts // Initialize MetaClient and Stack online
webpack.config.js // webpack configuration file
```
