### 客户端-上报性能数据

#### 编译使用方法
```
cargo build -p perf-report-tool --release
// perf-report-tool -n [agent_name] -t [testcaseId] -i [interval time (ms)]
./perf-report-tool -n sn-miner -t sn_perf1 -i 5000
```
#### 上报数据内容
```
pub struct BDTTestSystemInfo {
    pub name : String,
    pub testcaseId : String,
    pub cpu_usage: f32, // 所有和总的cpu 使用率 总100%。
    pub total_memory: u64, //机器总内存 单位 KB
    pub used_memory: u64, //使用内存 单位 KB
    pub received_bytes: u64, //下行速度 单位Bytes
    pub transmitted_bytes: u64,//上行速度 单位Bytes
    pub ssd_disk_total: u64, // ssd 容量
    pub ssd_disk_avail: u64,
    pub hdd_disk_total: u64, // hhd 容量
    pub hdd_disk_avail: u64,
}
```
#### 实现方式

```
    （1）使用 cyfs-util SYSTEM_INFO_MANAGER.get_system_info() 获取机器信息
    （2）使用http 上传node-test-data 服务，存储数据到mysql
    （3）使用nodejs echarts 绘图 
```
